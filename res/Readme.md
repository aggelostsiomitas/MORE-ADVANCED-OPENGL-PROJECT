The res folder contains all the necessary Shaders and Vertexe Shaders to run the 4 main.cpp files.

The Shaders folder constains the FragmentShader_11.glsl and the VertexShader_11.glsl that are needed for the main1.cpp file and the 
FragmentShader_12.txt, VertexShader_12.txt are for the main2.cpp

The Shaders_2 folder constains the FragmentShader_13.txt and the VertexShader_13.txt to run the main3.cpp file
and the FragmentShader_21.txt,VertexShader_21.txt,FragmentShader_Light_21.txt,VertexShader_Light_21.txt for the main4.cpp file
